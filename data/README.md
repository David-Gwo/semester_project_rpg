The recommended folder structure for a dataset is

	dataset_root/

		training/

			exp_1/
				images/
				labels.txt (optional)

			exp_2/
				images/
				labels.txt (optional)

			exp_n/
				images/
				labels.txt (optional)

		testing/

			exp_1/
				images/
				labels.txt (optional)

			exp_2/
				images/
				labels.txt (optional)
			exp_n/
				images/
				labels.txt (optional)
